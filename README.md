![mxcolr](./assets/gifcast_220130114850.gif)

Usage
=====
```
cp ./tmux ~/.config/tmux -r
```

```
tmux -f ~/.config/tmux/.tmux.conf -T 256 new-session\; attach \; source-file ~/.config/tmux/meta.min.tmuxtheme
```
